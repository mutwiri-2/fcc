Use viewport units to adjust size of elements

viewport width(vw) - sets width as a %ge of the width of the viewport of device used

viewport height(vh) - sets height as a %ge of the height of the viewport of device used

viewport minimum(vmin) - sets size of an element as a %ge of the smallest of the viewport measurements(width or height) of the viewport of device used

viewport maximum(vmax) - sets size of an element as a %ge of the largest of the viewport measurements(width or height) of the viewport of device used
